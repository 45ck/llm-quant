# 5.3 Inflation Regime x Risk Regime: 2x2 Allocation Matrix

> **Domain**: Macro Strategy & Regime Detection | **Priority**: P3 | **Status**: Pending

---

## System Context

This research targets **llm-quant**, a systematic macro paper-trading system managing a $100,000 portfolio across 39 assets spanning six asset classes:

- **US equity ETFs** (17): SPY, QQQ, IWM, DIA, VTI, IVV, XLK, XLF, XLE, XLV, XLI, XLY, XLP, XLU, XLRE, XLB, XLC
- **International equity** (3): VGK, EEM, EFA
- **Fixed income** (6): TLT, IEF, SHY, LQD, HYG, TIP
- **Commodities** (3): GLD, SLV, USO
- **Crypto** (5): BTC-USD, ETH-USD, SOL-USD, XRP-USD, ADA-USD
- **Forex** (5): EURUSD=X, GBPUSD=X, USDJPY=X, AUDUSD=X, USDCHF=X

**Technology stack**: Python 3.12, Polars DataFrames, DuckDB persistent storage, Pydantic configs.

**Current inflation awareness**: Zero. The system has no inflation-related indicators, no breakeven rate tracking, no CPI surprise measurement, and no commodity supercycle detection. The current regime model (`src/llm_quant/brain/context.py`) classifies markets as `risk_on`/`risk_off`/`transition` using VIX and a TLT/SHY yield spread proxy. Inflation is not a dimension of the regime space.

**Current commodity/TIPS positioning**: GLD, SLV, and USO are tradeable but the LLM receives only technical indicators (SMA, RSI, MACD, ATR) for these assets. TIP (TIPS ETF) is in the universe but there is no breakeven rate computation. Position sizing follows the same rules as all assets: max 2% per trade, max 10% per position. There is no inflation-aware allocation template.

**Current context pipeline**: `scripts/build_context.py` outputs JSON with `macro` section containing only `vix`, `yield_spread`, and `spy_trend`. No inflation fields exist. The LLM must infer inflation conditions from raw price data alone, which it does poorly and inconsistently.

**Risk limits** (file: `src/llm_quant/risk/limits.py`): max 2% NAV per trade, 10% per position, sector concentration < 30%, gross exposure < 200%, cash reserve >= 5%, max 15% drawdown. These limits are regime-agnostic -- they do not tighten in stagflationary environments or relax in benign inflation.

**Performance targets**: Sharpe > 0.8, max drawdown < 15%, Sortino > 1.0, Calmar > 0.5, annualized return 8-15%.

**Dependency**: This brief requires the risk regime output from Brief 5.1 (4-Factor Regime HMM) as one axis of the 2x2 matrix. If 5.1 is not yet implemented, the current 2-factor regime model can serve as a fallback, collapsing to a simpler risk_on/risk_off x inflation_up/inflation_down matrix.

## Research Question

Does integrating a dedicated inflation regime dimension -- constructed from TIPS breakeven rates, CPI surprise data, and commodity momentum -- alongside the existing risk sentiment regime produce a 2x2 allocation matrix that improves risk-adjusted returns on inflation-sensitive assets (GLD, SLV, USO, TIP, XLE, XLRE) compared to the current regime-agnostic allocation?

Specifically: does the 2x2 matrix correctly identify stagflationary episodes (risk_off + inflation_up) where traditional 60/40 portfolios suffer most, and does the corresponding "stagflation" allocation template (overweight commodities and gold, underweight equities and nominal bonds) preserve capital during these periods?

---

## Hypothesis

**H1 (2x2 matrix adds value)**: A 2x2 regime matrix combining risk sentiment (from Brief 5.1 or the current model) and inflation direction produces four economically distinct allocation environments:

| | **INFLATION UP** | **INFLATION DOWN** |
|---|---|---|
| **RISK ON** | "Reflationary Boom" -- overweight equities + commodities, underweight nominal bonds | "Disinflationary Boom" -- overweight growth equities + long duration, underweight commodities |
| **RISK OFF** | "Stagflation" -- overweight gold + commodities + TIPS, maximum underweight equities + nominal bonds | "Deflationary Bust" -- overweight long duration (TLT) + cash, underweight everything else |

Expected Sharpe improvement: >= 0.1 over the risk-only regime model, driven primarily by:
- Improved commodity timing (avoiding commodity longs during deflation)
- Improved bond duration management (avoiding TLT during rising inflation)
- Capital preservation during stagflationary episodes (2022 H1 is the canonical example)

**H2 (Inflation regime construction)**: A simple two-factor inflation indicator will reliably classify inflation direction:
- **Factor 1**: 3-month change in 5-year TIPS breakeven rate (T5YIE). Positive change = rising inflation expectations.
- **Factor 2**: 3-month momentum of WTI crude oil (DCOILWTICO). Positive momentum = supply-side inflation pressure.
- **Rule**: `INFLATION_UP` if both factors are positive. `INFLATION_DOWN` if both are negative. `INFLATION_NEUTRAL` if mixed (treated as the prevailing state from last observation for simplicity).

This simple rule captures 70%+ of inflation regime changes within 1 month, based on the historical relationship between breakeven rates and subsequent CPI prints.

**H3 (Stagflation allocation)**: The "Stagflation" template (overweight GLD/SLV/USO/TIP, underweight SPY/QQQ/TLT) will outperform the naive "risk_off" allocation (which overweights TLT) by >= 5 percentage points during identified stagflation periods, because nominal bonds lose value when inflation rises even as equities fall.

**Null hypothesis**: Inflation regime adds no incremental information beyond risk sentiment for a daily-rebalanced portfolio. The additional complexity of a 2x2 matrix (4 templates instead of 3) creates more false signals than useful allocation shifts.

## Methodology

### Phase 1: Inflation Data Acquisition (Week 1)

1. **Extend the FRED data fetcher** from Brief 5.1 (or build a new one if 5.1 is not started). Fetch daily series:
   - `T5YIE` -- 5-Year Breakeven Inflation Rate (TIPS-implied)
   - `T10YIE` -- 10-Year Breakeven Inflation Rate (cross-validation)
   - `CPIAUCSL` -- CPI-U All Items (monthly, seasonally adjusted)
   - `PPIACO` -- PPI All Commodities (monthly, seasonally adjusted)
   - `DCOILWTICO` -- WTI Crude Oil Spot Price (daily)
   - `GOLDAMGBD228NLBM` -- Gold Fixing Price (London, daily)

2. **Construct CPI surprise series**:
   - Download historical CPI releases and consensus forecasts from FRED or alternative free source
   - CPI surprise = actual CPI MoM - consensus forecast
   - This is a supplementary validation signal, not a primary regime input (because CPI is monthly and lagging)

3. **Build inflation regime indicator** (weekly frequency, matching Brief 5.1):
   - `breakeven_3m_change`: T5YIE current minus T5YIE 63 trading days ago
   - `wti_3m_momentum`: (WTI current / WTI 63 days ago) - 1
   - Classification rule:
     - `INFLATION_UP`: breakeven_3m_change > 0 AND wti_3m_momentum > 0
     - `INFLATION_DOWN`: breakeven_3m_change < 0 AND wti_3m_momentum < 0
     - `INFLATION_NEUTRAL`: mixed signals (carry forward previous state)

4. **Historical validation**: Check inflation regime labels against realized CPI changes with 3-month and 6-month lag. Compute precision/recall of `INFLATION_UP` label for predicting subsequent CPI YoY > 3%.

### Phase 2: 2x2 Matrix Construction (Week 2)

1. **Combine risk regime and inflation regime**:
   - Risk regime: from Brief 5.1 (4 states: risk_on, credit_stress, dollar_squeeze, risk_off) or current model (3 states: risk_on, risk_off, transition)
   - Simplify risk axis to binary for the 2x2: risk_on (includes risk_on and transition with SPY above SMA200) vs. risk_off (includes risk_off, credit_stress, dollar_squeeze, and transition with SPY below SMA200)
   - Cross with inflation axis: INFLATION_UP vs. INFLATION_DOWN
   - Result: 4 regime quadrants

2. **Define allocation templates** (initial weights, to be refined but not optimized on test data):

   **Reflationary Boom** (risk_on + inflation_up):
   - Equities: 45% (tilt toward XLE, XLB, XLI -- cyclical/value)
   - Commodities: 20% (GLD 10%, USO 5%, SLV 5%)
   - Fixed Income: 15% (TIP 10%, SHY 5% -- short duration, inflation-protected)
   - International: 10% (EEM 5%, VGK 5%)
   - Cash: 10%

   **Disinflationary Boom** (risk_on + inflation_down):
   - Equities: 55% (tilt toward XLK, QQQ -- growth/tech)
   - Fixed Income: 25% (TLT 15%, IEF 10% -- long duration benefits from falling inflation)
   - Commodities: 5% (GLD only, as hedge)
   - International: 10% (EFA 5%, VGK 5%)
   - Cash: 5%

   **Stagflation** (risk_off + inflation_up):
   - Commodities: 25% (GLD 15%, SLV 5%, USO 5%)
   - Fixed Income: 20% (TIP 15%, SHY 5% -- inflation-protected, short duration)
   - Equities: 10% (XLE 5%, XLP 5% -- energy and staples only)
   - Forex: 5% (USDCHF=X as safe haven)
   - Cash: 40%

   **Deflationary Bust** (risk_off + inflation_down):
   - Fixed Income: 40% (TLT 25%, IEF 10%, SHY 5% -- maximum duration)
   - Commodities: 5% (GLD only, as crisis hedge)
   - Equities: 0%
   - Cash: 50%
   - Forex: 5% (USDJPY=X short as yen strengthens in deflation -- note: requires short capability or proxy)

3. **Compute regime-conditional return statistics** on training data (2005-2018):
   - For each of the 4 quadrants, compute: mean return, volatility, Sharpe, max drawdown for each of the 39 assets
   - Validate that asset-class return profiles differ meaningfully across quadrants (ANOVA or Kruskal-Wallis test, p < 0.05)
   - Check: do commodities genuinely outperform in INFLATION_UP quadrants? Does TLT genuinely lose value in INFLATION_UP + RISK_OFF?

### Phase 3: Portfolio Optimization within Templates (Week 3)

1. **Ledoit-Wolf shrinkage estimator**: For each quadrant, estimate a covariance matrix using the Ledoit-Wolf shrinkage estimator (from `sklearn.covariance.LedoitWolf`), which is better conditioned than the sample covariance for our 39-asset universe with limited observations per regime.

2. **Constrained mean-variance optimization** (MVO) within each template:
   - Objective: maximize Sharpe ratio
   - Constraints: weights must sum to template's asset-class allocations (e.g., equities sum to 45% in Reflationary Boom), individual position <= 10% (matching `risk/limits.py`), cash >= template floor
   - Use `scipy.optimize.minimize` with SLSQP solver
   - The MVO determines how to distribute within asset classes (e.g., how much SPY vs. QQQ vs. XLE within the 45% equity bucket)

3. **Walk-forward optimization**:
   - Recalibrate Ledoit-Wolf covariance estimates quarterly (every 13 weeks)
   - Training window: expanding, minimum 3 years
   - Implementation lag: 1 week (regime signal Friday, trade Monday)
   - Transaction costs: 10 bps per leg

4. **Backtest the 2x2 strategy on test set (2019-2025)**:
   - Weekly rebalancing to target template weights
   - Full friction model (costs, lag, cash drag)
   - Compare against:
     - Benchmark 1: 60/40 SPY/TLT
     - Benchmark 2: Risk-only regime model (Brief 5.1 or current 2-factor) with no inflation dimension
     - Benchmark 3: Static "All-Weather" inspired allocation (25% equities, 40% bonds, 15% commodities, 7.5% gold, 12.5% cash)

### Phase 4: Integration Design (Week 4)

1. **Context enrichment**: Add to `build_context.py` JSON output:
   - `inflation_regime`: string ("inflation_up", "inflation_down", "inflation_neutral")
   - `breakeven_5y`: float (current T5YIE level)
   - `breakeven_3m_change`: float (basis points)
   - `wti_3m_momentum`: float (percentage)
   - `regime_quadrant`: string ("reflationary_boom", "disinflationary_boom", "stagflation", "deflationary_bust")
   - `target_allocation`: dict mapping asset classes to target weights for current quadrant

2. **Design regime-aware risk adjustments**: When in "Stagflation" quadrant:
   - Reduce `max_gross_exposure` from 200% to 100%
   - Increase `min_cash_reserve` from 5% to 15%
   - Set `max_position_weight` for equities to 5% (from 10%)
   - These adjustments are configured in `config/regime_rules.toml`, not hardcoded

3. **Document the LLM prompt update**: The system prompt must explain the 4 quadrants so the LLM can reason about allocation within the template constraints. Add a section to the Jinja2 decision prompt template showing the current quadrant, its allocation targets, and which assets are favored/disfavored.

## Literature Review

### Primary References

1. **Erb & Harvey (2006)**, "The Strategic and Tactical Value of Commodity Futures," *Financial Analysts Journal*, 62(2), 69-97.
   - **Extract**: Comprehensive analysis of commodity futures returns across inflation regimes. Key finding: commodities have positive expected returns during unexpected inflation (CPI surprise > 0) and negative returns during unexpected deflation. Their "roll return" decomposition explains why long-only commodity exposure (like USO) differs from futures-based commodity indices. Critical for understanding why our ETF-based commodity exposure (GLD, SLV, USO) may behave differently from the futures-based results in the literature. Use their inflation-conditional return estimates to calibrate our Reflationary Boom and Stagflation templates.

2. **Gorton & Rouwenhorst (2006)**, "Facts and Fantasies about Commodity Futures," *Financial Analysts Journal*, 62(2), 47-68.
   - **Extract**: Documents that commodity futures are positively correlated with inflation and inflation surprises, negatively correlated with equities and bonds. Their finding that commodities are the only traditional asset class with positive inflation beta provides the economic rationale for overweighting commodities in INFLATION_UP quadrants. Key data: commodities returned +7% annualized during unexpected inflation periods vs. -4% during unexpected deflation. Also documents that the commodity-equity correlation is time-varying and regime-dependent.

3. **Ilmanen (2011)**, *Expected Returns: An Investor's Guide to Harvesting Market Rewards*, Wiley, Chapters 12-14.
   - **Extract**: Chapter 12 covers the inflation factor premium across asset classes. Chapter 13 covers commodity return drivers (spot return, roll yield, collateral return, rebalancing return). Chapter 14 covers real assets and inflation protection strategies. Key framework: the "macro factor" decomposition of asset returns into growth surprise, inflation surprise, and discount rate components. Use this framework to validate that our 2x2 matrix captures the two most important macro factors. Ilmanen's finding that TIPS breakeven rates are the best real-time inflation forecast supports our use of T5YIE as the primary inflation regime indicator.

4. **Ledoit & Wolf (2004)**, "A Well-Conditioned Estimator for Large-Dimensional Covariance Matrices," *Journal of Multivariate Analysis*, 88(2), 365-411.
   - **Extract**: The Ledoit-Wolf shrinkage estimator for covariance matrices, essential for portfolio optimization with more assets (39) than observations per regime (50-200 weeks). The estimator shrinks the sample covariance toward a structured target (identity matrix or single-factor model), improving condition number and reducing estimation error. Their formula for the optimal shrinkage intensity is implemented in `sklearn.covariance.LedoitWolf`. This is critical because naive sample covariance with 39 assets is severely rank-deficient in short regime windows.

5. **Bridgewater Associates (2012)**, "The All Weather Story," public white paper (available at bridgewater.com).
   - **Extract**: The conceptual framework for the 2x2 regime matrix originates here: Bridgewater's All Weather strategy allocates across four economic environments defined by growth x inflation expectations. Their insight that risk parity across the four environments produces robust long-term returns motivates our template-based approach. Key difference: our system is dynamic (shifting allocations based on regime identification) rather than static (equal risk across all environments). Use their 4-environment framework as the theoretical benchmark, but note that our system can outperform by concentrating in the identified regime rather than diversifying across all four.

### Supplementary References

6. **Ang (2014)**, *Asset Management: A Systematic Approach to Factor Investing*, Oxford University Press, Chapter 7.
   - **Extract**: Factor-based framework for inflation hedging. Documents that equities are a poor short-term inflation hedge but a reasonable long-term hedge. TIPS are the most direct inflation hedge but have low expected excess returns. Commodities provide inflation hedging but with high volatility. Use this framework to set realistic expectations for the inflation-sensitive portion of the portfolio.

7. **Campbell, Sunderam & Viceira (2017)**, "Inflation Bets or Deflation Hedges? The Changing Risks of Nominal Bonds," *Critical Finance Review*, 6(2), 263-301.
   - **Extract**: Documents that the bond-stock correlation flipped from positive (1960s-1990s, when inflation was the dominant macro risk) to negative (2000s-2010s, when deflation was the fear). This regime change in the bond-stock correlation is crucial for our model: the "Deflationary Bust" template relies on negative bond-equity correlation (long TLT hedges short equities), but this relationship is not guaranteed in high-inflation environments.

8. **Neville, Draaisma, Funnell & Harvey (2021)**, "The Best Strategies for Inflationary Times," *Journal of Portfolio Management*, 47(8), 8-37.
   - **Extract**: Systematic study of asset class and strategy performance across inflation regimes from 1926-2021. Key finding: trend-following strategies (links to Brief 5.2) and commodity futures are the best inflation hedges. Equities and nominal bonds both suffer in high-inflation periods. Gold is an effective hedge only in extreme inflation (>5% CPI). Use their historical performance tables to calibrate our template weights.

## Evaluation Criteria

### Success Thresholds (all measured on test set 2019-2025)

| Criterion | Threshold | Measurement |
|---|---|---|
| Sharpe improvement vs. risk-only regime | >= 0.1 | Annualized Sharpe on 2x2 portfolio vs. risk-only regime portfolio |
| 2022 H1 drawdown improvement | >= 5 percentage points vs. 60/40 | Max drawdown during Jan-Jun 2022 |
| Stagflation period return | >= 0% annualized (capital preservation) | Return during identified stagflation quadrant weeks |
| Inflation regime accuracy | >= 65% of INFLATION_UP labels followed by rising CPI within 3 months | Precision of inflation_up label |
| Commodity allocation timing | GLD/SLV/USO contribute positive return in INFLATION_UP, near-zero in INFLATION_DOWN | Per-quadrant conditional return attribution |
| Template Sharpe by quadrant | >= 0.3 in each quadrant | Conditional Sharpe within each of the 4 quadrants |
| MVO improvement over equal-weight | >= 0.05 Sharpe improvement from Ledoit-Wolf MVO vs. equal-weight within templates | Intra-template optimization value |

### Failure Conditions

- 2x2 model Sharpe is lower than risk-only regime model (inflation dimension adds noise)
- The inflation regime oscillates frequently (average duration < 4 weeks), creating excessive turnover with no signal value
- Stagflation allocation loses money during identified stagflation periods (the template is wrong)
- CPI surprise has no correlation with the breakeven-based inflation label (the indicator is disconnected from reality)
- Ledoit-Wolf MVO produces extreme or unstable weights that require frequent manual override
- Total annual turnover exceeds 600% (the strategy is too expensive to implement even in a paper portfolio)

## Anti-Overfitting Warnings

### Specific Pitfalls for This Research

1. **Template weight optimization on test data**: The allocation templates (45% equity in Reflationary Boom, etc.) must be set before looking at test-set performance. If templates are adjusted after seeing 2019-2025 results, the backtest is contaminated. **Mitigation**: Set templates using only training-set (2005-2018) conditional return statistics and economic reasoning. Document the template weights and their justification before running the first test-set simulation.

2. **Inflation regime overfitting to 2022**: The 2022 inflation shock is the most dramatic inflation event in our sample. There is a strong temptation to calibrate the inflation indicator to detect 2022 perfectly. But 2022 is just one episode in a 7-year test set. **Mitigation**: Report results both including and excluding 2022. If the strategy's value disappears outside of 2022, it is overfitted to a single event.

3. **Look-ahead bias in breakeven rates**: TIPS breakeven rates can be computed in real time from market data (they are just bond prices). However, some FRED series have publication lags. **Mitigation**: Verify that T5YIE values are available same-day on FRED. If there is a 1-day lag, shift all signals by 1 day to ensure no look-ahead.

4. **Small sample of inflation regimes**: The test period (2019-2025) contains perhaps only 2-3 distinct inflation regime changes (2020 deflation, 2021-2022 inflation, 2023+ disinflation). This is too few events for statistical confidence. **Mitigation**: Use the full 2005-2025 sample for regime classification accuracy metrics (precision/recall), even though portfolio performance is only evaluated on the 2019-2025 test set. Acknowledge the small-sample limitation explicitly.

5. **Covariance estimation error in short regimes**: Ledoit-Wolf improves on sample covariance but still needs enough observations per regime. If a quadrant appears for only 8-10 weeks, the 39x39 covariance estimate is unreliable. **Mitigation**: Set a minimum observation count (52 weeks) for running MVO. Below this threshold, use equal-weight within the template's asset-class buckets. Report how often this fallback triggers.

6. **Survivorship bias in commodity ETFs**: USO had a dramatic negative roll yield period in 2020 (oil contango). GLD and SLV have expense ratios that reduce returns vs. spot prices. Historical commodity futures performance (from the literature) will overstate what our ETF-based portfolio achieves. **Mitigation**: Backtest using actual ETF returns (from Yahoo Finance), not hypothetical commodity spot/futures returns. Report the "implementation gap" between theoretical commodity returns and realized ETF returns.

7. **Inflation beta instability**: The inflation sensitivity of equities, bonds, and commodities is not constant. In the 2000-2020 period, bond-equity correlation was negative (deflation fear dominated), but in 2022 it turned positive (inflation fear dominated). The template weights that work for one correlation regime may fail in another. **Mitigation**: Test whether the 2x2 allocation templates are robust to changes in the bond-equity correlation sign. If not, the templates may need to be correlation-regime-conditional (making the model a 2x2x2 cube), which significantly increases complexity and should be flagged as a future research direction rather than implemented now.

8. **HARKing via quadrant labeling**: Naming the quadrants ("Reflationary Boom," "Stagflation," etc.) before seeing the data is acceptable. But if the quadrant labels are adjusted after seeing which assets perform well in each state, this is Hypothesizing After Results are Known. **Mitigation**: Commit the quadrant definitions and template weights to the git repository before running any test-set analysis. Any subsequent changes must be documented as modifications from the original specification.
